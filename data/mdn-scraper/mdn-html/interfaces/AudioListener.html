This is an experimental technology
Because this technology's specification has not stabilized, check the compatibility table for the proper prefixes to use in various browsers. Also note that the syntax and behavior of an experimental technology is subject to change in future versions of browsers as the spec changes.

The AudioListener interface represents the position and orientation of the unique person listening to the audio scene. All PannerNode spatialize in relation of the AudioListener stored in the AudioContext.listener attribute.

It is important to note that there is only one listener per context and that it isn't an AudioNode.

Properties

Inherits properties from its parent, AudioNode.

AudioListener.dopplerFactor
Is a double value representing the amount of pitch shift to use when rendering a doppler effect. Its default value is 1.
AudioListener.speedOfSound
Is a double value representing the speed of sound, in meters per second. Its default value is 343.3 m/s and is used to calculate the doppler shift (but no propagation delay is automatically applied to a sound far from the listener).

Methods

Inherits methods from its parent, AudioNode.

AudioListener.setPosition()
Defines the position of the listener. The three parameters x, y and z are unitless and describes the vector in the 3D space using the right-hand Cartesian coordinate system. PannerNode objects use this position relative to individual audio sources for spatialization. The default value of the position vector is (0, 0, 0).
AudioListener.setOrientation()
Defines the orientation of the listener. It consists of two vectors, the first one, the front vector, defined by the three unitless parameters x, y and z, describes the direction of the face of the listener, that is the direction the nose of the person is pointing to; the second vector, the up vector, defined by three unitless parameters xUp, yUp and zUp, describes the direction of the top of the head. Both vectors must be separated by an angle of 90°, in linear analysis terms, they must be linearly independant. The front vector default value is (0, 0, -1) and the up vector default value is (0, 1, 0).
AudioListener.setVelocity()
Defines the velocity vector of the listner. The velocity of an audio source relative to the listener's is used do control the pitch change needed to conform with the Doppler effect due to the relative speed. The vectors controls both the direction of travel, and its intensity, as the three parameters x, y and z are expressed in meters per second. The default value of the position vector is (0, 0, 0).
