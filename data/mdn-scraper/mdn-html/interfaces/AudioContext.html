This is an experimental technology
Because this technology's specification has not stabilized, check the compatibility table for the proper prefixes to use in various browsers. Also note that the syntax and behavior of an experimental technology is subject to change in future versions of browsers as the spec changes.

The AudioContext interface represents an audio-processing graph built from audio modules linked together, each represented by an AudioNode

An audio context controls the creation of the nodes it contains and the execution of the audio processing, or decoding.

An AudioContext can be target of events, therefore it implements the EventTarget interface.

Properties

AudioContext.destination Read only
Returns an AudioDestinationNode representing the final destination of all audio in the context. It often represents an actual audio-rendering device.
AudioContext.sampleRate Read only
Returns a float representing the sample rate, in samples per second, used by all node in this context. This limitation means that sample-rate converters are not supported.
AudioContext.currentTime Read only
Returns a double representing an ever-increasing hardware time using for scheduling. It starts at 0 and cannot be stopped, paused or reset.
AudioContext.listener Read only
Returns the AudioListener used for 3D spatialization.

Methods

Also implements methods from the interface EventTarget.

AudioContext.createBuffer()
Creates an AudioBuffer.
AudioContext.createBufferSource()
Creates an AudioBufferSourceNode.
AudioContext.createMediaElementSource()
Creates an MediaElementAudioSourceNode associated with an HTMLMediaElement associated with elements like <video> or <audio>.
AudioContext.createMediaStreamSource()
Creates an MediaStreamAudioSourceNode associated with a WebRTC MediaStream representing an audio stream, that may come from the local computer microphone or other sources.
AudioContext.createMediaStreamDestination()
Creates an MediaStreamAudioDestinationNode associated with a WebRTC MediaStream representing an audio stream, that may be stored in a local file or sent to another computer.
AudioContext.createScriptProcessor()
Creates an ScriptProcessorNode.
AudioContext.createAnalyser()
Creates an AnalyserNode.
AudioContext.createGain()
Creates an GainNode.
AudioContext.createDelay()
Creates an DelayNode.
AudioContext.createBiquadFilter()
Creates an BiquadFilterNode.
AudioContext.createWaveShaper()
Creates an WaveShaperNode.
AudioContext.createPanner()
Creates an PannerNode.
AudioContext.createConvolver()
Creates an ConvolverNode.
AudioContext.createChannelSplitter()
Creates an ChannelSplitterNode.
AudioContext.createChannelMerger()
Creates an ChannelMergerNode.
AudioContext.createDynamicsCompressor()
Creates an DynamicsCompressorNode.
AudioContext.createOscillator()
Creates an OscillatorNode.
AudioContext.createWaveTable()
Creates an WaveTableNode.
