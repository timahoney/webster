The HTMLAnchorElement interface represents hyperlink elements and provides special properties and methods (beyond those of the regular HTMLElement object interface they also have available to them by inheritance) for manipulating the layout and presentation of such elements.

Properties

Inherits properties from its parent, HTMLElement and from URLUtils.

Name Type Description
accessKey DOMString A single character that switches input focus to the hyperlink.
charset Obsolete DOMString The character encoding of the linked resource.
coords Obsolete DOMString Comma-separated list of coordinates.
download DOMString Indicates that the linked resource is intended to be downloaded rather than displayed in the browser. The value represent the proposed name of the file. If the name is not a valid filename of the underlying OS, browser will adapt it.
hash DOMString The fragment identifier, including the leading hash mark ('#'), if any, in the referenced URL.
host DOMString The hostname and port (if it's not the default port) in the referenced URL.
hostname DOMString The hostname in the referenced URL.
href DOMString Reflects the href HTML attribute, containing a valid URL of a linked resource.
hreflang DOMString Reflects the hreflang HTML attribute, indicating the language of the linked resource.
media DOMString Reflects the media HTML attribute, indicating the intended media for the linked resource.
name Obsolete DOMString Anchor name.
pathname DOMString The path name component, if any, of the referenced URL.
port DOMString The port component, if any, of the referenced URL.
protocol DOMString The protocol component, including trailing colon (':'), of the referenced URL.
rel DOMString Reflects the rel HTML attribute, specifying the relationship of the target object to the link object.
relList Read only DOMTokenList Reflects the rel HTML attribute, as a list of tokens.
rev Obsolete DOMString Reverse link type.
search DOMString The search element, including leading question mark ('?'), if any, of the referenced URL
shape Obsolete DOMString The shape of the active area.
tabindex long The position of the element in the tabbing navigation order for the current document.
target DOMString Reflects the target HTML attribute, indicating where to display the linked resource.
text DOMString Same as the Node.textContent property.
type DOMString Reflects the type HTML attribute, indicating the MIME type of the linked resource.

Methods

Inherits methods from its parent, HTMLElement, and from URLUtils.

Name & Arguments Return Description
blur() void Removes keyboard focus from the current element.
focus() void Gives keyboard focus to the current element.

These two methods are inherited from HTMLElement from HTML5 on, but were defined on HTMLAnchorElement in DOM Level 2 HTML and earlier specifications.

Specifications

Specification Status Comment
WHATWG HTML Living Standard Living Standard The following property has been added: download.
Technically, the URL-related properties, media, host, hostname, pathname, port, protocol, search, and hash, have been moved to the URLUtils interface, and HTMLAreaElement implements this interface.
HTML5 Candidate Recommendation The methods blur() and focus(), as well as the properties tabindex and accessKey, are now defined on HTMLElement.
The following properties are now obsolete: charset, coords, name, rev, and shape.
The following properties have been added: hash, host, hostname, media, pathname, port, protocol, relList, search, and text.
Document Object Model (DOM) Level 2 HTML Specification Recommendation No change from Document Object Model (DOM) Level 1 Specification.
Document Object Model (DOM) Level 1 Specification Recommendation Initial definition.
